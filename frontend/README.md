# AI Career Architect - Frontend Prototype

This is the frontend prototype for the AI Career Architect platform, specifically focusing on **Stage 1: The Recruiter (Discovery)**.

## Features (Page 1)

- **Avant-Garde UI**: Futuristic, dark-themed design with "intentional minimalism".
- **Hero Section**: Branding and introduction.
- **Recruiter Agent Interface**:
  - **CV Upload**: Drag-and-drop zone simulation.
  - **Analysis Simulation**: Visual feedback for "parsing" and "matching".
  - **Interactive Chat**: Mocked conversation with the Recruiter Agent to confirm role and skills.

## Tech Stack

- **Framework**: React + TypeScript + Vite
- **Styling**: Tailwind CSS v3
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Fonts**: Michroma (Display) & Manrope (Body)

## How to Run

1.  Install dependencies:
    ```bash
    npm install
    ```
2.  Start the development server:
    ```bash
    npm run dev
    ```
3.  Open [http://localhost:5173](http://localhost:5173) in your browser.
